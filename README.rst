**How to run the app**

Once you cloned the repository, create a virtual environment for `python 3.7`,
and then do inside of the `mapwit` top folder::

  $ pip install -r requirements.txt
  $ export FLASK_APP=mapwit
  $ export FLASK_ENV=development
  $ flask run
